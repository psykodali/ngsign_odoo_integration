🖋️ NGSIGN Integration for Odoo

License: LGPL-3   |   Odoo Version: 18.0   |   Python Version: 3.10+

Professional electronic signature integration for Odoo Sales, powered by the NGSIGN platform.

📋 Overview

Transform your sales workflow with secure, legally-compliant electronic signatures. This module seamlessly integrates the NGSIGN electronic signature platform directly into Odoo, enabling you to send quotations and sales orders for digital signing with just one click.

Perfect for businesses that need to collect signatures from customers quickly and securely, whether they're individuals or companies with multiple contacts.

✨ Key Features

  • 🚀 One-Click Signature Requests: Send sales orders for signature directly from Odoo — no PDF exports or external tools.
  • 👥 Smart Contact Management: Choose specific company contacts to sign via an intuitive wizard.
  • ✏️ Inline Information Editing: Update signer email and phone numbers on the fly.
  • 📝 Flexible Signature Templates:
    • Define exact X/Y signature coordinates
    • Select pages (specific or last)
    • Support for Simple, DigiGO, and Choose Later types
    • Mark templates as default
  • 🔄 Resend Capability: Resend requests with full tracking and timestamps.
  • 📊 Complete Tracking & Status: Real-time monitoring (Draft, Sent, Signed, Expired, Cancelled).
  • 📥 Automatic Document Download: Signed files automatically attached to Sales Orders.
  • 🔒 Secure & Compliant: Qualified signature compliant with international e-signature standards.

📦 Installation

Prerequisites

  • Odoo 18.0+
  • Python 3.10+
  • PyPDF2 library for PDF processing

Installation Steps

pip install PyPDF2
cd /path/to/odoo/addons
git clone https://github.com/yourusername/ngsign_integration_v4.git

Restart Odoo, update the Apps list, then install "NGSIGN Integration".

⚙️ Configuration

  1. API Credentials Setup:
    • Settings → General Settings → NGSIGN Integration
    • Base URL: https://ngsign.app (Production) or https://sandbox.ng-sign.com (Sandbox)
    • Bearer Token: your NGSIGN API key
  2. Create Signature Templates:
    • Settings → NGSIGN → Signature Templates
    • Configure coordinates, page placement, and signature type
    • Mark default templates for quick access

🎯 Usage

  1. Open a Sales Order (in Draft or Sent state)
  2. Click "Send with NGSIGN" (yellow button with signature icon)
  3. Select the signer, edit contact info if needed
  4. Choose a signature template and send
  5. Monitor signature status directly on the sales order:
    • 🔵 Sent
    • 🟢 Signed
    • 🟠 Expired
    • ⚫ Cancelled
  6. Signed document automatically attaches to the order and posts in the chatter

🔧 Technical Details

Technical Name: ngsign_integration_v4
Version: 18.0.1.0.0
Category: Digital Signature
License: LGPL-3
Dependencies: base, mail, sale

Database Models

  • ngsign.signature.template – defines signature placement, type, and default behavior.
  • ngsign.signer.wizard – transient wizard for signer and template selection.
  • sale.order (extended) – adds fields for transaction UUID, status, and signed document link.

API Endpoints

  • POST /server/protected/transaction/pdfs
  • POST /server/protected/transaction/{uuid}/launch
  • GET /server/any/transaction/{uuid}
  • GET /server/any/transaction/{uuid}/pdfs/{identifier}

🐛 Troubleshooting

  • PyPDF2 not installed: run pip install PyPDF2
  • 401 Unauthorized: check and refresh your Bearer Token
  • Button not visible: ensure Sales Order is in Draft/Sent state
  • Signed document missing: manually click "Check Signature Status"
